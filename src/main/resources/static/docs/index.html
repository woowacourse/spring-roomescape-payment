예약 검색 API

주어진 조건에 맞는 예약들(예약 대기 제외)을 조회합니다.

요청

GET /admin/reservations?memberId=2&themeId=1&dateFrom=2024-04-01&dateTo=2024-05-01 HTTP/1.1
Accept: application/json
Host: localhost:50638
Cookie: token=eyJhbGciOiJIUzI1NiJ9.eyJzdWIiOiIxIiwiaWF0IjoxNzE3OTk4MTc0LCJleHAiOjE3MTgwMDE3NzR9.Z6eCRvQ-PVcZLlIGyt0JKeDJyvEWgJMGtV207aQz3Bg

쿼리 파라미터 정보입니다.

Parameter Description

memberId

검색할 사용자 아이디입니다.

themeId

검색할 테마 아이디입니다.

dateFrom

검색하고 싶은 예약의 시작 날짜입니다.

dateTo

검색하고 싶은 예약의 끝 날짜입니다.
